# Sistema de Pesquisa de Saúde Mental Corporativa

## Sobre o Projeto

Sistema web desenvolvido para monitorar e analisar o bem-estar psicológico de colaboradores em ambientes corporativos através de questionários digitais anônimos.

### Funcionalidades Principais

- **Gestão Multiempresa**: Suporte a múltiplas empresas com isolamento de dados
- **Questionários Anônimos**: Coleta de dados sobre saúde mental preservando a privacidade dos colaboradores
- **Banco de Perguntas**: Sistema flexível para criação e gerenciamento de questionários

### Perfis de Usuário

- **Administrador**: Gerencia empresas, usuários e questionários

## Tecnologias

- **Frontend**: React + TypeScript + Vite
- **UI**: shadcn-ui + Tailwind CSS
- **Autenticação**: JWT
- **Arquitetura**: Multitenant

## Como Executar

```sh
# Clonar o repositório
git clone <YOUR_GIT_URL>

# Instalar dependências
npm i

# Iniciar servidor de desenvolvimento
npm run dev
```
